# dotfiles
dotfiles for hyprland arch linux configuration

## packages configured
kitty (terminal)\
oh-my-posh (cmd line styling)\
waybar (taskbar)\
wofi (app launcher)\
fastfetch (fetch)\
neovim (code editor)

pipewire (audio)\
zathura (pdf reader)\
optimus manager (gpu control)\
